# Gorstan v3.2.0 – Fred Stack Edition

## 🚀 How to Run Locally
```bash
npm install
npm run dev
```

## 🛠️ How to Build
```bash
npm run build
```

## 👾 Godmode Commands
- `/doors` – Show all exits
- `/goto [roomID]`
- `/get [itemID]`
- `/godhelp` – Show all commands

## 🌟 Features
- Trait system
- Traps and puzzles
- NPCs with memory and mood
- Intro sequence with coffee logic
- Reset Room countdown
- Secret endings
